## #005: fleur de eLan（フルールドゥエラン）茅場町

**サイト:** https://www.elan-v.com
**mood:** 温もりミニマル・上質カジュアル・スローラグジュアリー
**quality_score:** 5/5

### 配色 — ウォームトーン体系
全色に「温かみ」が入ってる。冷たいグレーが一切ない。

**テキスト色（4段階）:**
- #080808 / #121212（最も濃い・メイン見出し）
- #3c4043（本文寄り）
- #817d7d（薄め・補助テキスト）
- **#a9a39c**（ウォームグレー）← **9回使用、最多。**このサイトの色の核

**背景色（4段階のウォーム白）:**
- #FFFFFF / #fff（純白）
- **#fcfaf7**（ほんのり温かい白）← 3回
- **#faf7f3**（もう少し温かい白）← 4回
- **#eae5db / #e9e6db**（ウォームベージュ）← セクション背景

**ボーダー:**
- #f1efe9（ほぼ見えないベージュボーダー）
- #a9a39c（ウォームグレー）
- #5b5a5a（濃いめ）

**注目:** 全部の色に黄色〜ベージュの色味が入ってる。#faf7f3は「白」だけど純白#fffとは明確に違う。**この「ちょっとだけ温かい白」がサイト全体の空気を作ってる。** ウチのテンプレは全部純白#fffだった。

### フォント
| 用途 | フォント | weight | 特徴 |
|------|----------|--------|------|
| 英語見出し | **Josefin Sans** | **100**（極細）, 300, 400 | 幾何学サンセリフ。w100は針金みたいに細い |
| 英語サブ | Roboto | 400 | |
| 日本語全般 | 游ゴシック Medium | - | body指定 |

**注目: Josefin Sans weight: 100**
font-weight: 100 は「Thin」「Hairline」と呼ばれる最も細いウェイト。これを大きなサイズで使うと、繊細さと余裕が同居する。**太字で強調するのではなく、細字で引き算する**アプローチ。Google Fontsで無料。

### タイポグラフィ
- body: font-size: **13px** / line-height: **1.84**
- line-heightが**全部微妙に違う**: 1.57, 1.62, 1.73, 1.84, 2.12（5段階で精密調整）
- letter-spacing: 0.05em, 0.06em, 0.12em, 0.14em（控えめ。La Meluの0.4emとは対照的）

**注目: line-heightの精密さ**
1.57と1.62、1.73と1.84。人間の目には僅かな差やけど、各セクションの「空気感」がこの0.1の違いで変わる。プロは「全部1.8でええやろ」とは言わない。**コンテンツに合わせてline-heightを微調整してる。**

### レイアウト
- body: `min-width: 1024px`（**PCファーストで設計**。スマホ対応は別途ある可能性）
- body: `display: flex; flex-direction: column; min-height: 100vh`
- コンテンツ幅: 76%を多用（px固定ではなく%で余白を確保）
- GSAP使用（スクロールアニメーション）
- ローディングアニメーションあり

### トランジション（独自イージング多数）
- `cubic-bezier(.215, .61, .355, 1.2)` → **1.2超え=バウンス感**。行き過ぎてから戻る動き
- `cubic-bezier(0.39, 0.575, 0.565, 1)` → 滑らかなease-out系
- `cubic-bezier(0.075, 0.82, 0.165, 1)` → 急加速→ゆっくり止まる
- `transform 0.25s ease-out` / `transform 0.3s ease-out`

**注目:** cubic-bezierの第4引数が1.0を超えてる（1.2）。これは意図的な「オーバーシュート」で、自然な物理挙動に近くなる。

### ホバー
- 特殊なhoverなし。Tailwind系のbg-opacity変化のみ
- La Meluと同じく**控えめ派**

### steal_point TOP3
1. **ウォーム白 #faf7f3 / #fcfaf7** → 純白#fffの代わりに使うだけでサイト全体が温かくなる。ウチのテンプレ全部に即適用できる。コスト0で印象激変
2. **Josefin Sans weight: 100** → 極細フォントの引き算デザイン。見出しを「太くして目立たせる」のではなく「細くして余裕を見せる」。Google Fontsで無料
3. **line-heightの精密5段階制御** → 1.57/1.62/1.73/1.84/2.12。ひとつの値で全部済ませるんやなくて、コンテンツに合わせて微調整する姿勢がプロ
